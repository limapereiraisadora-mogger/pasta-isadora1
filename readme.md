# 🌾 Agrinho: Jornada do Alimento — Do Campo à Mesa

Este é um projeto interativo desenvolvido para o **Programa Agrinho**, com o objetivo de conscientizar as pessoas sobre o papel da tecnologia e da sustentabilidade no agronegócio moderno. 

O site apresenta, de forma visual e dinâmica, o caminho que o alimento percorre desde o momento do plantio até chegar às nossas mesas, destacando práticas que protegem o meio ambiente e evitam o desperdício.

---

## 🚀 Funcionalidades

* **Painel Interativo:** O usuário pode clicar em diferentes etapas da produção agrícola (Plantio, Colheita, Transporte e Consumo).
* **Conteúdo Educativo:** Cada etapa exibe informações sobre o uso de tecnologias (como drones e GPS) e dicas de sustentabilidade.
* **Design Responsivo:** Interface simples, limpa e adaptável para computadores e celulares.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais do desenvolvimento web (Front-End), sem a necessidade de frameworks externos:

* **HTML5:** Para a estruturação dos elementos e acessibilidade da página.
* **CSS3:** Para a estilização, identidade visual (temática verde do campo) e efeitos de animação nos cards.
* **JavaScript (Vanilla):** Para a lógica de manipulação do DOM, permitindo que as informações apareçam dinamicamente na tela ao clicar.

---

## ⚙️ Como Executar o Projeto

Como o projeto utiliza apenas arquivos estáticos, você não precisa instalar nenhuma dependência. 

1. Faça o clone deste repositório ou baixe os arquivos.
2. Abra a pasta do projeto no seu computador.
3. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador de preferência.
